Basic Page Structure
  • <!DOCTYPE html>

    • The first element in HTML document is the <!DOCTYPE>, which is required for browsers to display webpages correctly. It is an example of an empty element, an element that does not need a closing tag.

  • <html>

    • The next element is the <html> element, the root element. It contains a start and end tag that concludes the document.

  • <head>

    • After the <html> element, the <head> element follows with a start and end tag. This element contains meta information about the document, such as the <title> element that defines the document’s title.

  • <body>

    • The last basic document structure is the <body> element. The visible page content is contained within this element.

Example:

<!DOCTYPE html>

<html>

<head>

<title>Webpage Title</title>

</head>

<body>

<h1>Hello World!</h1>

</body>

</html>

Headings/Paragraphs

Headings

Headings help to define a documents structure and make it easier to read. They are used by search engines to catalog the structure and content of a web page. Heading tags run from the most important <h1> to the least <h6> and include a start and end tag.

Heading Examples:

  • <h1>Heading 1</h1>
  • <h2>Heading 2</h2>
  • <h3>Heading 3</h3>
  • <h4>Heading 4</h4>
  • <h5>Heading 5</h5>
  • <h6>Heading 6</h6>

Paragraphs

The element that defines a paragraph in HTML is the <p> element. The display of a paragraph can be unpredictable. Adding extra spaces will not changes the results displayed, the browser removes any extra spaces or lines. In order to preserve spaces and lines, the <pre> element is used.

Paragraph Example:

  • Code:

    • <p>This is an example paragraph.

    An example paragraph is this.

    A paragraph as an example.</p>

  • Display:

    • This is an example paragraph. An example paragraph is this. A paragraph as an example.

Preserve Paragraph Example:

  • Code:

    • <pre>This is a preserved example paragraph.

    A preserved paragraph example.

    Paragraph with spaces and lines preserved.</pre>

  • Display:
    • This is a preserved example paragraph.
A preserved paragraph example.
Paragraph with spaces and lines preserved.
Links/Images

Links

A HTML link, or hyperlinks, can be clicked on to move to another document. It can link to another HTML element or image. The <a> tag defines a link with the href attribute specifies the destination address for the link. For example, <a href=”url”>link</a>. CSS can be used to change the color behavior of a link.

Link CSS Options:

  • a:link – used for normal links
  • a:visited – visited links
  • a:hover – the mouse hovers over the link
  • a:active – a link as it is clicked.

Images

Images can enhance the design of a webpage. HTML defines images with the <img> tag, which is an empty tag and contains attributes only. The src attribute specifies the URL of an image. The other required attribute is the alt attribute that provides alternate text if an image can’t display. Optional attributes include the width and height attributes used to specify their respective categories. Images can also be used as links.

Normal Image:

<img src =”url” alt=”alt text”>

Image Link:

<a href=”url1”><img src=”url2” alt=”alt text”></a>
Lists

HTML provides the option to create ordered or unordered lists.

Unordered lists

  • Unordered lists start and end with the <ul> tag.
  • Each list item starts and ends with the <li> tag.
  • Items by default will be marked with bullets or discs. Use list-style-type to define the style of the list item markers, such as circles or no markers.
  • Example:

    • <ul style=”list-style-type:square;”>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li></ul>

Ordered lists

  • Ordered lists start and end with the <ol> tag.
  • Each list item starts and ends with the <li> tag.
  • Items by default with be marked with numbers. Use the type attribute to define the type of list item marker, such as letters or roman numerals.
  • Example:

    • <ol type=”A”>

    <li>Item 1</li>

    <li>Item 2</li>

    <li>Item 3</li></ol>

Tables

A table, or structure for creating rows and columns on page, in HTML is defined by the <table> tag. The <tr> tag defines table rows, the <th> tag defines table headers, and the <td> tag defines table data/cells. In order for them to stand out, table headings are bold and centered by default. The colspan attribute can be added to make a cell span more than one column. The rowspan attribute will make a cell span more than one row. The <caption> tag adds a caption to the table.

Example:

<table>

<caption>Table caption</caption>

<tr>

<th>First heading</th>

<th colspan=”2”>Second heading</th>

</tr>

<tr>

<td>Cell 1</td>

<td>Cell 2</td>

<td>Cell 3</td>

</tr>

</table>

CSS can be used to set table properties.

  • Border and border-collapse properties set table borders.
  • The padding property sets table cell padding.
  • Use text-align to align the text in headings or cells.
  • Border-spacing can be used to set the spacing of the table border.
